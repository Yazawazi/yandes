## 这玩意能做啥子

## 这玩意的一些特点
* 使用 API 获取图片，不再是分析页面代码来 GET 到图片。
* 恰 Python2.7 的饭。
* 好像没了。。。
## 怎么用这玩意
* 首先，你确保你有 Python2.7 的环境。
* 然后， `git clone https://github.com/Yazawazi/yandes`。
* 最后， `python yande.py`。
* 若为 Linux 系统，请使用 `python2 yande.py`。
